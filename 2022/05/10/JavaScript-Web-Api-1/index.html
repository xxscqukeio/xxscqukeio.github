DOM

2.1 DOM的构成

多层节点构成的文档 添加、删除、修改页面各部分

跨平台、语言无关的表示、操作网页

2.2 节点类型

节点层级

任何html、xml都可以表示为—–由节点构成的层级结构

表示为 特定节点为根的 树形结构

document 为根节点

唯一子节点html元素 文档元素

Node类型

JavaScript中所有的节点类型都继承Node类型 12个数值常量

开发者最常用的2种 元素节点、文本节点

1. nodeName与nodeValue

2者保存有关节点的信息

nodeName = 元素标签名 nodeValue = null

2.节点关系

节点与其他节点关系 形容为家族关系

body是html 互为父子元素

head与body同胞元素

使用item()访问NodeList元素

item()方法 使用()或者[] 倾向使用中括号 类数组对象

使用Array.prototype.silce()将NodeList对象转换为数组

每个节点都有parentNode属性

Sibling同胞节点 第一个节点previousSibling 最后一个节点nextSibling

当childNodes只有一个节点 2者都是null

父节点第一个和最后一个子节点 专门属性

firstChild和lastChild

便利方法hasChildNodes()

true 说明节点有一个或者多个子节点

ownerDocument 指向代表 整个文档的 文档节点指针 所有节点都有共享关系

3.操纵节点

所有关系指针都是只读

最常用的方法appendChild() 在childNodes列表末尾添加节点

insertBefore(newCode,参照节点)方法

将newCode 插入到参照节点前的一个同胞节点

appendChild() 和 insertBefore(newCode,参照节点)方法

插入时，不会删除任何已有节点

replaceChild(插入节点，替换节点)

removeChild(移除的节点)

replaceChild和removeChild替代和移除的节点 仍然被同一个文档拥有

但文档内无其位置

4.其他方法

所有节点类型仍然共享了2个方法

cloneNode(true/false) 参数是否深复制

true参数 进行深复制—复制节点及其整个子DOM树

false参数 只会复制调用该方法的节点 复制返回节点为orphan孤儿节点

只复制HTML属性 不会复制JavaScript属性 例如事件处理程序

normalize() 唯一任务 处理文档子树中的文本节点

检测这个节点的所有后代

发现空文本节点—-删除

2个同胞节点相邻—-合并一个文本节点

Document

Document类型可以表示HTML或者XML文档

获取关于页面的信息及 操纵外观和底层结构

文档子节点

appendChild() removeChild() replaceChild()等不会用在document对象

文档类型 只读的 只能有element类型的子节点

文档信息

document属性URL、domain、referrer

只有domain属性可以设置

不能给domain设置URL中不包含的值

设置document.domain 解决不同子域页面 跨域通信的问题

将每个页面document.domain设置成相同的值 页面之间可以通信了

domain 属性的浏览器限制 属性一旦放松就不能收紧

定位元素

getElementById()和getElementsByTagName()

多个相同id元素 返回文档中出现的第一个元素

getElementsByTagName()

0个或多个元素的NodeList

HTMLCollection对象

id为了匹配对应的label元素

hasFeature()返回值并不可靠、废弃

文档写入

write()–writeln() 用于动态包含外部资源

write简单写入文本

writeln在字符串末尾追加一个换行符(“\n”)

不能直接添加字符串</script>输出后 匹配最外层的<script>

改成"<\/script>"不会匹配最外层的<script>

open()–close() 用于打开和关闭网页输出流

window.onload事件处理程序 将函数推迟到页面加载完后执行

element类型

html元素标签名始终全大写表示

xml(XHTML)始终与源代码中的大小写一致

不确定脚本文档类型中 将标签名转换为小写形式

html元素

标准属性:

id 元素在文档唯一标识符

title包含元素的额外信息 以提示条形式展示

lang 元素内容的语言代码（很少用”en”）

dir 语言的书写方向（”ltr”left to right “rlt” 很少用）

className（class属性 指定元素css类 class为ECMAScript 不能直接用）

取得属性

3个属性相关DOM方法

getAttribute() setAttribute() removeAttribute()

属性名不区分大小写

自定义属性名data-以方便验证

第二个属性 事件处理程序或者事件属性 onclick

设置属性

setAttribute()

在DOM对象上添加自定义属性 不会自动让它变成元素属性:

removeAttribute()

不单单是清除属性的值 完全将属性从元素中去掉
