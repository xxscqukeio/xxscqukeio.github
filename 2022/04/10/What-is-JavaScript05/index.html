What is JavaScript 5

Created2022-04-10|Updated2022-04-11
|Post View:

操作符与语句

操作数据

会调用 valueof() 和 toString() 方法

一元操作符

递增、递减操作符

无论前缀递增还是后缀递减 操作符

变量的值 都会在 语句被求值之前改变

单独使用 唯一操作 不会改变执行结果

和其他操作混合时 差异明显

一元加和减 + -

一元加 应用到非数值 回执行 Number()转型函数 数据转换 然后加上正负

位运算符

  1. 按位非 ~ 返回数值的补数

  2. 按位与 & 1&1=1 其他全为0

  3. 按位或 | 0|0全为0 其他为1

  4. 异或^ 相同返回0 相异返回1

  5. 左移 << 左移5个0

  6. 有符号右移 >>> 无符号 正数都相同

    负数有巨大差距 将符号位当做正数的二进制处理 （目前没碰到过 感觉有病）

布尔操作符

逻辑非 |

相等操作符

逻辑非操作符 可以将 任意值 转换为布尔值 !! =Boolean()

!转换为布尔值 然后取反

逻辑与 &&

短路特性

第一个操作数 决定了结果 后面就无效

逻辑或

避免赋值null或者undefined

let myObject = preferedObject || backupObject;

乘性操作符

乘法、除法、取模

NaN

Infinity / Infinity = NaN

指数操作符 **

Math.pow(3, 2) 3**2

sqrt ** = 0.5;

加性 + -

Infinity -Infinity = NaN

-Infinity- (-Infinity) = NaN

关系操作符 > < <= >=

字符串比较 对应编码的 数值

相等操作符 == != ===

null == undefined

null === undefined false

条件操作符 ? num1: num2;

(num1>num2) num1: num2;

赋值操作符 =

赋值

, 辅助声明多个变量

语句 流控制语句

if

do while 后测试 循环

退出前 至少执行一次

while 先测试 循环

可能不会执行

for语句 先测试

1.初始化

2.先测试

3.表达语句

for(;;){ //无穷循环

}

只包含条件表达式 = while 循环

for(;i<count;){

}

for-in 语句

用于枚举 对象中的 非符号键属性

对象的属性是 无序的 因此 不能保证返回对象属性的顺序

for-of 语句

遍历 可迭代对象的元素

es2018

for-await-of 支持生成 期约promise

标签语句

label: statement

break 和 continue 语句

标签使用描述性强的文本

嵌套不要太深

with语句

影响性能 难于调试

不推荐

switch语句

switch 每个条件 比较会 使用全等操作符 不会发生强制数据类型转化 “10” 不等于 10

函数

核心组件

可以封装语句 任何地方时间 执行

diff()计算差值 不同分支不同return

最佳实践：函数要么返回值、要么不返回值

只在特定条件下返回值的function 会在调试时 出现麻烦

严格模式下的函数 限制：

不能以 eval 、 arguments 名称

参数 不能 eval 、 arguments

2个 命名参数 不能 同名

小结

基本元素

  1. 7个基本数据类型 Undefined Null Boolean Number String Symbol BigInt + Object 复杂数据类型

  2. 严格模式 容易出错处 施加限制

  3. ECMAscript C类操作符

    函数与其他语言函数不一样

  4. 不需要指定函数返回值 可以在任何时候返回

  5. 不指定的返回值的 实际 返回 特殊值undefined
