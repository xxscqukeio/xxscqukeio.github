What is JavaScript 4

Created2022-04-08|Updated2022-04-09
|Post View:

数据类型

ECMAscript 6种原始类型 （简单类型）

undefined null number

boolean

string symbol

object bigInt

typeof 操作符 （运算符）

typeof null 返回的是 object

Undefined类型

当var或者let没有初始化–undefined 默认未初始化值为undefined

未赋值为undefined 未声明会报错

建议声明变量的同时进行初始化

Null类型

空指针对象 因此 typeof 会返回 object

定义保存对象的变量 null来初始化 – 空对象指针语义

undefined 与 null 表面 ==

但实际用途不同

Boolean类型

true false

number 0、NaN–false

Number类型

整数、浮点数(双精度)

八进制字面量 070–56 08–8

严格模式下无效

16进制 Ox

+0 和 - 0所有情况下等同

浮点值

有效但不推荐

可以采用科学计数法

浮点值精度最高可到17位小数 不要测试是否等于某个特定的浮点值

不能表示所有数值 最小值保存在 Number.MIN_VALUE = 5e-324

也存在Number.MAX_VALUE

非常大或小的数 有必要检测一下范围

NaN

0/任意数 = NaN

涉及到NaN始终返回NaN

true可以转换为1 不是NaN

但是其他字符串都是

字符串0001 转换 1

parseInt(“AF”,16); //175

parseFloat(“0xA”) ; //0

遇到的第一次出现的小数点有效，第二次就无效

String类型

“”,’’,``

字符串一旦创建 之不可变 字符串 必须销毁原始的字符串 包含在新的变量中

模板字面量

保留换行字符 可以跨行定义字符串

换行符 可以保持反引号内部的空格 注意缩进问题

字符串插值

${value}

标签函数tag function

使用String.raw标签函数获取原始字符 换行符或Unicode

不会被转换或转义

Symbol类型

es6新增类型

唯一 不可变

确保 对象属性 使用唯一标识符 不发生属性冲突的危险

创建唯一的记号 非字符串形式的对象属性

使用Symbol() 符号初始化 符号本身是原始类型

创建Symbol() 并作用于对象新属性 保证不会覆盖已有的对象属性

不能与new关键字一起作为构造函数使用

使用全局符号注册表

Symbol.for() 对每个字符串键等 执行幂等操作

关于内置符号。。。

symbol 类型用于创建对象的唯一标识符。我们在这里提到 symbol 类型是为了完整性，但我们要在学完 object 类型后再学习它。

Object类型

object 类型是一个特殊的类型。

其他所有的数据类型都被称为“原始类型”，因为它们的值只包含一个单独的内容（字符串、数字或者其他）。

相反，object 则用于储存数据集合和更复杂的实体。
