如何轻松写出正确的链表代码？

指针含义

c语言 – Java 引用

存储所指对象的内存地址

p->next=q

p节点next指针—-存储了q节点的内存地址

插入结点时，一定要注意操作的顺序—–>指针丢失、内存泄露

链表插入时：先将待插入节点挂到链表上，再将链表中前向的地址指向新插入的节点

  1. 结点 x ->next 指针指向结点 b
  2. 结点 a -> next 指针指向结点 x

删除链表结点时，也一定要记得手动释放内存空间
