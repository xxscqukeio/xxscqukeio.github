JavaScript特性

自动分号插入 换行符也被视为分隔符

代码块 {...} 后以及有代码块的语法结构（例如循环）后不需要加分号

有 8 种数据类型：

  • number — 可以是浮点数，也可以是整数，
  • bigint — 用于任意长度的整数，
  • string — 字符串类型，
  • boolean — 逻辑值：true/false，
  • null — 具有单个值 null 的类型，表示“空”或“不存在”，
  • undefined — 具有单个值 undefined 的类型，表示“未分配（未定义）”，
  • object 和 symbol — 对于复杂的数据结构和唯一标识符，我们目前还没学习这个类型。

模态框，它们会暂停代码执行并阻止访问者与页面的其他部分进行交互

如果任何一个操作数是一个字符串，那么另一个操作数也将被转换为字符串

?? 运算符选取值为已定义的值（defined value）的变量。a ?? b 的结果是 a，除非 a 为 null/undefined，这时结果是 b。

值 null 和 undefined 是特殊的：它们只在 == 下相等，且不相等于其他任何值。

局部变量 变量只在函数内部可见

参数可以有默认值：function sum(a = 1, b = 2) {...}。

如果没有 return 语句，那么返回的结果是 undefined。
