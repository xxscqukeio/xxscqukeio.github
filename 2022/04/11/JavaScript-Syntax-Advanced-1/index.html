函数进阶

内建函数的示例：

如 alert(message)、prompt(message, default) 和 confirm(question)。

局部变量

函数中声明的变量只在该函数内部可见

外部变量

对外部变量拥有全部的访问权限。函数也可以修改外部变量

如果在函数内部声明了同名变量，那么函数会 遮蔽 外部变量。

减少全局变量的使用是一种很好的做法。现代的代码有很少甚至没有全局变量。大多数变量存在于它们的函数中。但是有时候，全局变量能够用于存储项目级别的数据。

  • 参数（parameter）是函数声明中括号内列出的变量（它是函数声明时的术语）。
  • 参数（argument）是调用函数时传递给函数的值（它是函数调用时的术语）。

如果 text 参数未被传递，它将会得到值 "no text given"。

现代 JavaScript 引擎支持 空值合并运算符 ??

指令 return 可以在函数的任意位置。当执行到达时，函数停止，并将值返回给调用代码（分配给上述代码中的 result）

空值的 return 或没有 return 的函数返回值为 undefined

jQuery 框架用 $ 定义一个函数。LoDash 库的核心函数用 _ 命名。

自描述:函数名（isPrime）

函数表达式

某种意义上说一个函数是一个特殊值**

一个函数是表示一个“行为”的值

代码发生的细节：

  1. (1) 行声明创建了函数，并把它放入到变量 sayHi。
  2. (2) 行将 sayHi 复制到了变量 func。请注意：sayHi 后面没有括号。如果有括号，func = sayHi() 会把 sayHi() 的调用结果写进func，而不是 sayHi 函数 本身。
  3. 现在函数可以通过 sayHi() 和 func() 两种方式进行调用。

在 ask(...) 调用内进行函数声明。这两个函数没有名字，所以叫 匿名函数

函数在 ask 外无法访问

ask 的两个参数值 showOk 和 showCancel 可以被称为 回调函数 或简称 回调

函数表达式 vs 函数声明

函数声明：在主代码流中声明为单独的语句的函数。

严格模式下，当一个函数声明在一个代码块内时，它在该代码块内的任何位置都是可见的。但在代码块外不可见。

优先使用函数声明

声明无效时 使用函数表达式
